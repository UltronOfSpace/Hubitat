How This Works

This page makes a single HTTP request to your Hubitat hub to fetch all device data:

GET http://<your-hub-ip>/device/list/data

The hub returns a JSON array — a list of objects, one per device. Each object contains fields like:

{
  "id": 42,
  "displayName": "Living Room Lamp",
  "deviceTypeName": "Generic Zigbee RGBW Light",
  "deviceRoomName": "Living Room",
  "status": "ACTIVE",
  "lastActivityTime": "2026-04-17T21:25:43+0000",
  "zigbeeId": "00124B00251B581C",
  "type": "sys"      // "sys" = built-in driver, "usr" = user-installed driver
}

This page then parses that JSON and renders it as the table above. You could do the same thing in any programming language, home automation tool, or even a spreadsheet.

Try it: Enter your hub's IP address above and click "Load Devices" to see your own device list.

Note: This only works from your local network (the same network your hub is on). The hub does not require any login or password for this endpoint.
